# FVTT Shadowrun5E HUD 모듈

### 설치 방법
부가 모듈 -> 모듈 설치 -> 하단의 매니페스트 URL에 주소 입력 후 설치 -> 월드에서 Shadowrun 5e HUD 모듈 활성화

* 매니페스트： https://github.com/stevejew/SR5_HUD/releases/latest/download/module.json

### 사용방법
토큰을 우클릭 한 후 오른쪽 아래에 있는 폴더 모양의 아이콘을 누르면 작동합니다. <br>
캐릭터 초상화를 더블클릭하면 액터 시트를 열고 닫을 수 있고, 초상화 옆에 있는 방패 모양을 누를 경우 물리 피해 저항 판정이 가능합니다.
